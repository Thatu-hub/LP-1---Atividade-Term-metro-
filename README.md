##  Requisitos

Para executar os programas é necessário ter um **compilador C** instalado.

Recomendado: **GCC (GNU Compiler Collection)**

Para verificar se o GCC está instalado, execute no Prompt de Comando:

```
gcc --version
```

Se aparecer a versão do GCC, significa que ele está instalado corretamente.

---

##  Como executar os programas (Windows)

### Abrir o Prompt de Comando

Pressione **Windows + R**, digite:

```
cmd
```

e pressione **Enter**.

---

### Navegar até a pasta do exercício

Use o comando `cd` para navegar entre pastas.

Exemplo:

```
cd nome_da_pasta
```

Para voltar uma pasta:

```
cd ..
```

Para listar os arquivos da pasta atual:

```
dir
```

Continue navegando até encontrar a pasta que contém o arquivo `main.c`.

---

###  Compilar o programa

Quando estiver na pasta que contém `main.c`, execute:

```
gcc main.c -o programa
```

Esse comando criará um arquivo executável chamado **programa.exe**.

---

###  Executar o programa

```
programa
```

O programa será executado no Prompt de Comando.
